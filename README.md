# Food Delivery Application

A full-stack Food Delivery Web Application developed using Java, Servlets, JSP, MySQL, and Next.js.

## Features

- User Registration & Login
- Browse Restaurants
- Browse Food Categories
- Restaurant Menu
- Add to Cart
- Update Cart
- Checkout
- Place Orders
- Order History
- Offers
- Favorites
- Help Center

## Technologies Used

### Backend

- Java
- JDBC
- Servlets
- Apache Tomcat

### Frontend

- Next.js
- React
- TypeScript
- Tailwind CSS

### Database

- MySQL
